# XivelyThis is a repo for managing machine readable artifacts for the [Xively APIs](https://xively.com/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Xively APIs](https://xively.com/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://xively.com/)- [Developer](https://xively.com/)- [Twitter](https://twitter.com/XivelyIOT)- [GitHub](https://github.com/xively)The [APIs.json](https://github.com/api-evangelist/xively/blob/master/apis.json) index provides links to all the relevant links for the Xively APIs, including the OpenAPI and Postman if they are present.
